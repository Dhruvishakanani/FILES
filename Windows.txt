
------------------------
  Command Prompt (cmd)
------------------------
  Create folder
--> mkdir FOLDER_NAME
--> md FOLDER_NAME

  Delete folder 
--> rmdir FOLDER_NAME
--> rd FOLDER_NAME

  Show file/folder
--> dir

  Rename file/folder
--> ren OLD_NAME NEW_NAME
  
  Check any program version
-->PROGRAM_NAME --version

  Clone website using cmd
--> wget --mirror --convert-link --adjust-extension --page-requisites --no-parent HTTPS://WEBSITE_NAME.DOMAIN
--> wget --mirror --convert-links --wait=2 HTTPS://WEBSITE_NAME.DOMAIN

  Show connected wifi network password
--> netsh wlan show profile
--> netsh wlan show profile name=PROFILE_NAME key=clear

  Create battery report
--> powercfg/batteryreport

  Show upgradation of software 
--> winget upgrade
--> winget upgrade --all

  Other
--> notepad FILE_NAME.EXTENSION:PASSWORD
--> wmic bios get serialnumber
